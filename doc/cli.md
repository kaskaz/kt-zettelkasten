# ktzk-cli

## The command-line interface

### Commands
* ___new [--name | -n] [--tag | -t]___

    options:
    * name - Is the title of the note and the name of the file. Expected to be specified between quotation marks.
    * tag - Can have multiple tags

* ___read [--id | -i]___

    options:
    * id
     
* ___update [--id | -i]___

    options:
    * id
    
* ___delete [--id | -i]___

    options:
    * id
    
* ___list [--last]___
